1. (18 баллов) Реализовать класс Complex (не case):
a. Два apply метода через два Double числа и через String.
b. toString, equals, hashCode
c. Арифметические операции +, -, *, /, unary_-, unary_+, ^ (возведение в комплексную степень)
d. unapply метод, чтобы в паттерн матичнги работало
e. Методы Re, Im, conjugation, sqrt, abs.
f. Реализовать implicit конверсию из Double и Int в Complex.

2. (8 баллов) Реализовать две implicit конверсии A => B, B => C, так чтобы из них следовала конверсия A => C

3. (5 баллов) Реализовать экстрактор Both, который проверяет два паттерна:
1 match {
  case Both(x, y) =>
}
В данном случае оба x и y будут 1.
